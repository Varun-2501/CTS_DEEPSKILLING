# CTS Deepskilling Hands-on

This repository contains the mandatory and additional important hands-on
folder structure from the DotNet FSE spreadsheet.

Folder names are organized as:

```text
Skill/
  Mandatory hands-on name/
```

Repeated skill names and repeated hands-on names within the same skill are
included only once. Colons in exercise names are represented by hyphens
because Windows does not allow colons in folder names.
